Bart Bash - Godot Game Files
==============================

To run the game:
1. Open neighbor.html in a web browser
2. The game should load automatically

Files:
- neighbor.html: Main game page
- neighbor.js: Godot engine JavaScript
- neighbor.wasm: WebAssembly binary
- neighbor.pck: Game data package
- neighbor.audio.worklet.js: Audio processing
- load.jpg: Loading screen background
